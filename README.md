# Hunter-s-solution
This is a house hunting platform that aims at minimizing time and energy, by helping our clients find houses at the comfort of their homes. It is a responsive multi-page real estate website developed using HTML5 and CSS3. The website is designed to help users browse available rental properties, learn about the company, explore services, and access user authentication pages.

## Features

- Responsive navigation bar
- Hero sections with background images
- Property listings with featured homes
- About Us page
- Services page
- Login and Registration pages
- Contact and Schedule Viewing sections
- Customer Reviews section
- Responsive layouts using CSS Grid and Flexbox
- Reusable and organized CSS components

## Technologies Used

- HTML5
- CSS
- Google Fonts (Poppins)
- Font Awesome Icons
